## Cara Jalankan 4B
- Buka foder 4B pada terminal
- Jalankan "npm install"
- Buat Database dengan nama "wikigames" di pgadmin
- Jalankan "npx sequelize-cli db:migrate"
- Jalankan "npm run dev"
- Buka browser "localhost:5000"
